# ⚙개발 환경
* ``C#``
* ``Unity 2022.3.42f1``
* ``Rider``
* ``Git Fork``

# 🕹기능 구현 
1. 몬스터 움직임 구현
2. 영웅 공격 구현
3. HP바 구현
4. 체력시스템 구현 

# ✏️사용 기술
1. 어드레서블을 사용한 리소스 관리
2. 몬스터 상속 구조로 관리 
3. Raycast를 사용하여 탐지하고 RigidBody2D로 몬스터 움직임구현 
4. ResourceManager로 프리팹을 로드 => ObjectManager로 로드된 오브젝트들 생성 및 관리 => PoolManager에서 오브젝트풀로 관리
